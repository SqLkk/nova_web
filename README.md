# Supernova Web (nova_web) 🌐

This repository contains the web & GitHub Pages interface for the Supernova platform, including mock factory databases and browser-based SCADA visualization UI.

## 📁 Repository Structure

- **`ui/`**: Angular web application configured for static / GitHub Pages deployment.
- **`create_factory_db.py`**: Helper script for building mock factory SQLite database.
- **`factory_mock.sqlite`**: Pre-generated factory dataset for offline/demo operation.

## 🛠️ Setup & Running

```bash
cd ui
npm install
npm start
```
